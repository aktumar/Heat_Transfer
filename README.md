# Numerical Solution of Heat Transfer (OpenGL)

> RICHARDSON`S METHOD

$$
ψ_(i,j)^(k+1)=1/2(1+β^2 )  (〖ψ_(i+1,j)^k+ψ_(i-1,j)^k  +β^2 ψ_(i,j+1)^k+β^2 ψ_(i,j-1)^k-〖Δx〗^2 w〗_(i,j)  )
$$
